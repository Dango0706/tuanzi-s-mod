# Tuanzi's Mod 1.0.2 Changelog

## New Features
- **New Special Item: Immortal Talisman**:
  - **Core Function**: While this item is in the player's inventory (including backpack, hotbar, offhand, or armor slots), the player will not drop any items upon death and will retain all experience points.
  - **Safety**: The item itself is not dropped on death and is perfectly retained on the player after respawning.
  - **Independent Mechanism**: Uses low-level Mixins to directly intercept drop logic. It does not rely on "Soulbound" enchantments, avoiding potential conflicts with other death-triggering mods.
  - **Obtainment**: Currently has no survival recipe; accessible only via commands or Creative mode.

## Improvements
- **Version Update**: Mod version has been officially upgraded to 1.0.2.
