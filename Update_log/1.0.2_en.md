# Tuanzi's Mod 1.0.2 Change Log

## New Features
- **New Special Item: Immortal Talisman**:
  - **Core Function**: When in inventory, the player will not drop items and will retain all experience upon death.
  - **Mechanism**: Uses low-level Mixins to intervene directly, avoiding conflicts with Soulbound enchantments.
  - **Obtainment**: Commands or creative mode only for now.

- **New Potion: Undying Potion**:
  - **Effect**: Grants the 'Undying' status effect.
  - **Variants**: Includes normal, extended, splash, lingering, and tipped arrows.
  - **Registration**: All added to the creative inventory.

- **Warden Reward Boost**:
  - **XP Increase**: Experience dropped increased to **275**.
  - **Rare Drop**: Dealing >**50%** max health damage grants a **10%** (affected by Looting) chance to drop a **Soulbound** enchanted book.

- **New Enchantment: Experience**:
  - **Core Function**: Increases experience points gained from killing mobs.
  - **Effect**: Increases experience gain by **25%** per level (rounded up).
  - **Rules**: Only effective when held in the main hand; works on all mobs except players.
  - **Level**: Max level 4. Level 30 enchantment table yields up to level 3. Level 4 requires anvil merging or loot.
  - **Obtainment**: Tradeable and can be found in loot chests.

- **Enchantment Compatibility Boost**:
  - **Infinity + Mending**: The exclusivity between **Infinity** and **Mending** has been removed. They can now coexist on the same bow.
