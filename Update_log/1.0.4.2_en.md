# Tuanzi's Mod 1.0.4.2 Changelog

## Fixes

- **Fixed an issue where villager trades inside the Soul Merchant Station did not automatically unlock (restock) after players slept**:
  - **Fix**: Replaced the restock day tracking logic using `level.getGameTime()` (which measures cumulative elapsed server ticks and does not advance when sleeping) with `level.getOverworldClockTime()` (which tracks the overworld clock and correctly skips forward to morning when sleeping). Now, trades inside the Soul Merchant Station will properly unlock after players sleep through the night.

- **Fixed an issue where the Echo Breaker was not classified as a "sword"**:
  - **Fix**: Used Fabric DataGen to correctly add the Echo Breaker to the vanilla "swords" (`#minecraft:swords`) item tag. It now fully benefits from all sword-specific enchantment and targeting mechanisms.
