# Tuanzi's Mod 1.0.5 更新日志

## 新增

- **集成 Modrinth 自动更新检测器**:
  - **模组加载时异步检测**：在客户端初始化时异步发送请求至 Modrinth 接口，精准判断是否有新版本发布。
  - **主界面平滑弹窗与高阶视觉效果**：若检测到可更新版本且玩家未忽略，将在进入主界面时弹出高质感提示卡片。提供“自动更新”、“忽略更新一天”（以时间戳写入本地配置）和“暂不更新”选项。
  - **遮罩层交互防穿透悬停优化**：在弹窗绘制底层 `TitleScreen` 画面时，通过高精度的坐标屏蔽（传参 `-999, -999`），彻底解决了当鼠标悬停在弹窗空隙时，原本主界面的“单人游戏”等按钮仍会被选亮（Hover）的视觉穿透瑕疵，使 UI 拥有完美的阻断质感。
  - **暂不更新与忽略一天交互闭环修复**：将进程已通知状态（`alreadyNotified`）提到最先拦截顺位，彻底解决了因点击“暂不更新”或“忽略更新一天”返回主菜单时，由于重新执行 `TitleScreen#init` 触发 Mixin 重复弹回，导致界面陷入无限死循环而无法关闭的底层逻辑问题。
  - **Markdown 富文本日志渲染**：自主研发高性能极简 Markdown 解析器，完美将 Modrinth 返回的原始 Markdown 格式日志（含标题、圆点列表、`**加粗**`、`*斜体*`以及`` `亮橙黄色行内代码块` ``）直接翻译成 Minecraft 原生的富文本色彩与样式，极具尊贵感。
  - **多行裁剪平滑滚动与尊贵滚动条**：为更新日志引入像素级 Scissor 屏幕裁剪区和内存级折行缓存。当日志字数过多时，玩家可直接使用鼠标滚轮平滑地上下拖拽以阅读全部更新内容，且卡片右侧配有极其炫酷的**半透明白滑道与亮蔚蓝色滑块滚动条**，视觉交互质感拉满。
  - **安全无删自动升级**：玩家选择“自动更新”后，自动下载最新的 `.jar` 文件，并严格遵守 GEMINI.md 安全规范，将旧包重命名为 `.jar.bak`，完全避免直接删除文件带来的安全隐患。
  - **重启提示与安全退出**：下载完成后，友好提示玩家需要重启，并提供“立即重启”（安全退出游戏）和“稍后重启”选项。
  - **平滑动画网络故障提示**：若因网络原因检测失败，在主界面右上角优雅绘制淡入淡出的半透明红色警示框（500ms 渐入，最后 500ms 渐出），持续 10 秒后自动静默消失，不打扰玩家的正常操作。
  - **开发调试专属特权**：在 IDE 开发环境下运行模组时，强制跳过所有本地忽略和当次已通知标记，并具备自动断网兜底模拟功能，确保每次返回主菜单时始终完美弹出升级界面，极大便利开发人员进行调试。
  - **专用服务器物理隔离**：更新检测、UI 弹窗、自动下载升级等所有更新相关逻辑均强制限定于客户端（Client）运行，Dedicated Server（专用服务端）将彻底无视并忽略该检测过程，防止出现任何类加载冲突或服务端运行时崩溃。
