# Tuanzi's Mod 1.0.3 更新日志

## 新增功能
- **新增材料: 坚守者的心脏 (Warden Heart)**:
  - **稀有掉落**: 击杀监守者 (Warden) 时有 **25%** 的概率掉落（受到抢夺附魔影响，每级抢夺增加 10% 掉落率）。
  - **核心原料**: 是锻造究极防御武器“回响破障者”的必不可少的核心材料。
- **新增武器/防具: 回响破障者 (Echo Breaker)**:
  - **锻造合成**: 由 1 个 坚守者的心脏 与 3 个 回响碎片 和 1 根 木棍 在工作台合成。
  - **被动属性 (声波吸收)**: 仅在**主手持有**时生效。在面临监守者（Warden）音波爆破袭击时，**自动减免 30%** 的爆破伤害。
  - **主动技能 (完美格挡)**: 右键使用可提供持效 **0.5 秒** 的格挡防卫，且在**副手无物品**时才可以使用。格挡期间**免疫 100%** 监守者音波爆破伤害。
  - **完美反弹 (声波反射)**: 在完美格挡音波爆破成功的瞬间，会将致命声波完美折返反射，使发射源（监守者）立即受到 **15 点真实伤害** 并产生强烈的反冲退避击退。
  - **格挡代价**: 每次格挡成功时额外消耗 **3 点** 耐久值（共计消耗 **4 点**）。
  - **格挡冷却**: 若格挡**失败**（未拦截到音波伤害）则进入 **5.0 秒** 冷却时间；若格挡**成功**（完美折返反弹）则触发 **8.0 秒** 冷却时间。
  - **装备修复**: 可以使用**坚守者的心脏 (Warden Heart)**在铁砧中来修复这个物品。
- **新增附魔: 熔炼 (Smelting)**:
  - **核心功能**: 省去回家烧炉子的繁琐步骤。使用该工具挖掘“可在熔炉中烧炼的方块”时，直接掉落其烧炼后的产物。
  - **完美兼容**: 与 **时运 (Fortune)** 完美兼容，可以获得更多熔炼后的掉落物。
  - **惩罚机制**: 附魔产生的高温会加速工具磨损。每挖掘一个可熔炼方块，额外消耗 **1 点** 耐久（加上原本消耗，共计 **2 点**）。
  - **适用部位**: 镐、斧、锹。
  - **冲突机制**: 与 **精准采集 (Silk Touch)** 冲突，无法同时存在。
  - **获取方式**: 属于“宝藏附魔”，无法通过附魔台获得。
    - **遗迹探索**: 有概率出现在 **下界要塞** 或 **沙漠神殿** 的宝箱中。
    - **村民交易**: 极罕见地出现在 **大师级图书管理员** 的交易列表中（需 40-64 绿宝石）。
- **新增附魔: 连锁采掘 (Chain Mining)**:
  - **核心功能**: 按住快捷键激活（默认为 `~` 键），在挖掘方块时瞬间连锁破坏并采集周围相连的同类方块。
  - **开采上限**: 每级连锁 16 个方块，最高等级 IV 可连锁开采 64 个方块。
  - **耐久损耗**: 连锁破坏的每个方块正常扣除工具耐久。若在连锁中工具损毁，连锁会即刻安全切断。
  - **饱食度消耗**: 每额外挖掘 1 个方块，额外增加 **0.25** 消耗度。系统直接计算并即时扣除您的饱和度和饥饿值，免去 tick 判定延迟。
  - **激活提示**: 当连锁采掘激活（手持连锁附魔工具且按住快捷键）时，屏幕左上角会显示一个类似 F3 界面效果的灰色背景 HUD 提示，清晰直观。
  - **获取限制**: 目前暂无生存获取方式，仅限创造模式获取。

## 调整与优化
- **不死药水 (Undying Potion) 限制**:
  - 现在“不死”状态效果仅对 **玩家** 以及 **被玩家驯服的生物**（如驯服后的狼、猫等）生效。
  - 对其它野生生物或敌对生物不再起作用，防止意外“保护”了敌人。
