# Tuanzi's Mod 1.0.2 更新日志

## 新增功能
- **新增特殊物品: 不朽圣符 (Immortal Talisman)**:
  - **核心功能**: 当该物品存在于玩家物品栏（背包、快捷栏、副手、盔甲栏）中时，玩家死亡时消耗,使玩家不会掉落任何物品，且保留所有经验值。
  - **获取方式**: 目前暂无合成配方，仅可通过指令或创造模式获取。

- **新增药水: 不死药水 (Undying Potion)**:
  - **效果**: 赋予玩家“不死”状态效果。
  - **变体**: 包含普通、延长版，以及对应的喷溅药水、滞留药水和药水箭。
  - **注册**: 已全部加入创造模式物品栏。

- **Warden 奖励增强**:
  - **经验提升**: 掉落经验提升至 **275 点**。
  - **稀有掉落**: 若玩家对单个监守者造成超过 **50%** 生命上限的伤害，其死亡时有 **10%**（受抢夺影响）的概率掉落 **《灵魂绑定》** 附魔书。
- **新增附魔: 阅历 (Experience)**:
  - **核心功能**: 增加玩家击杀生物时获得的经验收益。
  - **效果**: 每一级增加 **25%** 的经验值（向上取整）。
  - **规则**: 仅在主手武器持有该附魔时生效；对除击杀玩家外的所有生物有效。
  - **等级**: 最高 4 级。30 级附魔台最高可附出 3 级，4 级需通过铁砧合并或特殊战利品获得。
  - **获取**: 可以被交易，可以从宝箱开出。

- **附魔兼容性调整**:
  - **无限 + 经验修补**: 解除了 **无限 (Infinity)** 与 **经验修补 (Mending)** 之间的互斥限制，现在它们可以同时存在于同一把弓上。
