# Tuanzi's Mod 1.0.2 更新日志

## 新增功能
- **新增特殊物品: 不朽圣符 (Immortal Talisman)**:
  - **核心功能**: 当该物品存在于玩家物品栏（背包、快捷栏、副手、盔甲栏）中时，玩家死亡不会掉落任何物品，且保留所有经验值。
  - **安全性**: 物品本身在死亡时也不会掉落，并在重生后原样保留在玩家身上。
  - **独立机制**: 采用底层 Mixin 直接干预掉落逻辑，不依赖“灵魂绑定”附魔，避免了与其他死亡触发类模组的潜在冲突。
  - **获取方式**: 目前暂无合成配方，仅可通过指令或创造模式获取。

- **新增药水: 不死药水 (Undying Potion)**:
  - **效果**: 赋予玩家“不死”状态效果。
  - **变体**: 包含普通、延长版，以及对应的喷溅药水、滞留药水和药水箭。
  - **注册**: 已全部加入创造模式物品栏。
