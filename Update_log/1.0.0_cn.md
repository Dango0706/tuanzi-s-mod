# Tuanzi's Mod 1.0.0 更新日志

## 新增功能
- **彩色海绵 (Color Sponge)**: 革命性的物品染色系统！
  - 在铁砧中使用彩色海绵作为副手材料，即可为物品重命名并应用丰富的色彩。
  - **十六进制支持**: 直接输入颜色代码（如 `#FF5555`）。
  - **传统代码**: 支持经典的 `&` 颜色代码（如 `&c`）。
  - **渐变色系统**: 支持炫酷的渐变效果（如 `&q3#FF0000-#0000FF`）。
  - **分割染色**: 支持通过特定格式实现多段颜色划分。
- **灵魂绑定附魔 (Soulbound)**: 保护你的珍贵装备！
  - 拥有此附魔的物品在玩家死亡时将保留在物品栏中，不会掉落。

## 系统调整
- **铁砧逻辑增强**: 深度重构了铁砧的命名处理机制，完美兼容彩色海绵的自定义格式。
- **自定义创造模式物品栏**: 新增 "Tuanzi's Mod" 标签页，方便快速获取模组物品及附魔书。

## 集成支持
- **JEI (Just Enough Items)**: 深度集成！
  - 可在 JEI 中直接查看彩色海绵的所有染色指令示例及其详细说明。
- **ModMenu**: 适配 Fabric 的模组菜单列表。
