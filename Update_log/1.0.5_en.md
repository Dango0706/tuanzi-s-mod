# Tuanzi's Mod 1.0.5 Changelog

## Features

- **Gacha System & Visual Gacha Pool Editor (Complete Overhaul, UX Upgrade & Fate Anchor)**:
  - **1. Fate Anchor (定轨) System**:
    - **Selection & Double Confirmation**: Players can left-click any item within the gacha pool preview GUI to set it as a fate target. A custom 9x3 read-only confirmation dialog will pop up showing the target and warnings, recording the selection into player states upon approval.
    - **Fate Points & Guaranteed Drop**: Obtaining an item of the same quality as the active fate target that is NOT the target itself grants +1 "Fate Point" (命定值), sending a chat message: "You obtained other xx quality item, Fate Point +1 (current x/y)". When fate points reach the maximum, the next item of this quality drawn is guaranteed to be the chosen target.
    - **Fate Limit Rules**: For the standard pool (`normal`), epic and above qualities (`legendary` / `epic`) have a limit of 2 points (guaranteed target on the 3rd roll), while qualities below epic have a limit of 1 point. For all other pools (treated as limited pools), all qualities have a maximum limit of 1 point.
    - **Cancel / Change target**: Players can clear their active fate target or select a different one at any time in the GUI. Canceling or changing target is quality-independent, meaning it only resets the active quality's fate points to 0, leaving other qualities' targets untouched.
    - **Rarity-Based Reset Rules**: When the targeted item is obtained (via normal luck or guaranteed drop): if the item's quality is epic or above, active fate points reset and the fate anchor ends (clears the target state). If the quality is below epic, only fate points are reset, and the fate anchor remains active.
    - **Red Diagonal Corner Arrows Visuals**: When a targeted item is detected in the gacha pool preview GUI, the game will render 4 red diagonal corner arrows pointing outward at the slot corners (NW, NE, SW, SE). This is drawn on top of its original quality-colored backgrounds, producing an outstanding Tech-hud target lock-on visual indicator.
    - **Write-Through Persistence & Crash Safety**: Any changes to active fate targets (setting/clearing) or fate values (pity rolls, duplicate rolls) now trigger immediate Write-Through synchronous disk flushes. Complemented by .tmp temporary atomic replacement writes (Atomic Move), this achieves ultimate crash and power-outage resistance. Furthermore, registered a SERVER_STOPPING lifecycle event hook to force-flush all online players' states before reboots or shutdowns, completely eradicating reboot data loss loops.
  - **2. Gacha Kernel Probability Core**:
    - Implementing sophisticated soft/hard pity logic for Legendary (normal starts rising at 60, guaranteed at 80; featured starts rising at 70, guaranteed at 90) and Epic (guaranteed every 20) items.
  - **3. Rapid Visual Gacha Pool Editor Suite (9x6 / 3x9)**:
    - Features single-click category tabs, main-hand anvil quick-additions, emerald jeweled hot-reloads, and bulk page-turning navigation, as well as hopper one-click backpack imports and a weight fine-tuning console. Supports delete lock with blinking warnings for right-click removals.
  - **4. Three-Dimensional Physics Gacha & Visual Ceremony**:
    - Single draws shoot a rocket forward; 10-folds launch a 10-rocket circle volley upward. Rewards inherit physics explosion speed and glow in rarity-colored scoreboard beacon outlines with owner-exclusive grab protections.
  - **5. Detailed History Tracking & HUD**:
    - Access complete transcripts via `/cardpool history` in 9x6 read-only GUI with warm rarity filled outlines, held-item automatic filter headers, and sneak-right-click HUD disclosure books.
  - **6. Standardized Diagnostic Logging**: All pity calculations and fate changes are written to the standard `ModLog.debug` framework.

  - **Damage Mechanic Details**: Every melee attack hit on a target with an enchanted sword triggers Abyssal Rhythm, dealing extra magic damage to the primary target. No cooldown, and the damage type is magic (bypasses physical armor).
