# Tuanzi's Mod 1.0.6 Changelog

## 26.2 Compatibility Upgrade & Bug Fixes
- Upgraded and adapted to Minecraft 26.2.
- Fixed compilation errors related to `EntityType` constant relocations to `EntityTypes`.
- Adapted and refactored stained glass panes and glazed terracotta references to the new `ColorCollection` API.
- Updated functional parameter interfaces for `PostSpawnProcessor` and `TeamColor`.
- Refactored client-side GUI hidden logic to use `hud.isHidden` and adapted to `Minecraft.setScreenAndShow`.
- Updated signature of the overridden `knockback` method in `LivingEntity`.
- Adjusted `OrderedSubmitNodeCollector.submitNameTag` parameter lists to remove the obsolete distance parameter.
- Successfully regenerated localizations, recipes, and tags using Fabric DataGen on version 26.2.
