# Tuanzi's Mod 1.0.4.2 更新日志

## 修复

- **修复灵笼贸易站内村民交易红叉无法在玩家睡觉后自动解除的问题**:
  - **修复**：将刷新判断的时间基准从 `level.getGameTime()`（游戏总运行 tick 数，睡觉不跳跃）修改为 `level.getOverworldClockTime()`（主世界时钟 tick 数，睡觉自动跳跃到清晨）。现在玩家睡觉跳过夜晚后，灵笼贸易站内的村民交易红叉将会正确地自动解除。

- **修复回响破障者不属于“剑”这一类的设定**:
  - **修复**：使用 Fabric DataGen 将回响破障者（Echo Breaker）正确归入原版“剑（`#minecraft:swords`）”物品标签中。现在它将完美享受所有与“剑”相关的附魔和判定机制。

- **修复 JEI 配方催化剂注册已弃用 API 警告问题**:
  - **修复**：将 [TuanzisJeiPlugin.java] 中已弃用的过时 API `addRecipeCatalyst` 替换为最新官方推荐的 `addCraftingStation(IRecipeType, ItemStack...)`。现在编译时不再会输出废弃 API 警告。
