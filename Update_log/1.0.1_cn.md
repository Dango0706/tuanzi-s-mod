# Tuanzi's Mod 1.0.1 更新日志

## 新增功能
- **突破重命名限制**: 彻底移除了铁砧中 50 字符的命名长度限制。现在你可以自由地使用长文本配合彩虹海绵的各种特效，创作更复杂的物品名称！
- **新增神级物品: 尤里的复仇**:
  - 拥有 5 点耐久度。
  - 它是世界上唯一可以破坏基岩的镐子。
  - 术业有专攻：它**只能**用来破坏基岩，无法破坏任何其他种类的方块。
  - 暂时无法通过常规生存手段获取。
- **新增状态效果: 飞行**:
  - **核心效果**: 获得该效果后，玩家在生存模式下也能像创造模式一样自由飞行。
  - **过期提醒系统**: 
    - 为了防止高空坠落，当飞行效果剩余 **15秒** 和 **10秒** 时，会通过 Actionbar 发送文字警告并播放提示音。
    - 当效果仅剩 **最后5秒** 时，将开启**每秒倒计时提醒**，伴随更高频率的音效反馈。
  - **配套药水**: 新增了飞行药水、喷溅型飞行药水、滞留型飞行药水及飞行药箭（包含 3 分钟基础版与 8 分钟延长版）。
  - **获取方式**: 目前暂无合成配方，可通过指令或创造模式获取。

## 物品调整
- **名称变更**: “颜色海绵”正式更名为 **“彩虹海绵”**，并同步更新了所有相关资源及 JEI 描述。
- **属性调整**: 彩虹海绵的堆叠上限从 64 下调至 **1**，使其更加珍贵。

## 修复
- **资源管理**: 转向全自动化的 Datagen 数据生成，解决了语言文件冲突及贴图丢失问题。
