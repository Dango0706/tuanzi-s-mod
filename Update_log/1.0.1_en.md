# Tuanzi's Mod 1.0.1 Changelog

## New Features
- **Removed Rename Limit**: Completely removed the 50-character limit for renaming items in the Anvil. You can now freely use long text with Rainbow Sponge effects!
- **New God-tier Item: Yuri's Revenge**:
  - Features 5 durability points.
  - The only pickaxe in the world capable of breaking Bedrock.
  - Specialized: It **only** breaks Bedrock and cannot destroy any other type of block.
  - Currently unobtainable through normal survival means.
- **New Status Effect: Flight**:
  - **Core Effect**: Allows players to fly in Survival mode just like in Creative mode.
  - **Expiration Alert System**: 
    - To prevent accidental falls, warnings are sent via Actionbar and sound effects when **15s** and **10s** remain.
    - During the **last 5s**, a **per-second countdown** is triggered with high-pitch sound feedback.
  - **Associated Potions**: Added Potion of Flight, Splash Potion, Lingering Potion, and Arrow of Flight (available in 3-min and 8-min versions).
  - **Obtainment**: Currently unobtainable via survival recipes; accessible via commands or Creative mode.

## Item Adjustments
- **Name Change**: "Color Sponge" has been officially renamed to **"Rainbow Sponge"**. All assets and JEI descriptions have been updated.
- **Stack Size**: The stack size of Rainbow Sponge has been reduced from 64 to **1**.

## Bug Fixes
- **Anvil Logic Fix**: Resolved server-side rename length validation issues for ultra-long names.
- **Attribute Fix**: Fixed a bug where "Yuri's Revenge" did not lose durability when breaking Bedrock.
- **Compatibility Fix**: Fixed Anvil Screen Mixin injection failures on Minecraft 1.21.2+.
- **Asset Management**: Migrated to full Datagen automation, resolving language file conflicts and missing textures.

## Improvements
- **Version Update**: Mod version has been officially upgraded to 1.0.1.
