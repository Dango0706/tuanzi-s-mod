# Tuanzi's Mod 1.0.0 Changelog

## New Features
- **Color Sponge**: A revolutionary item dyeing system!
  - Use the Color Sponge as a secondary material in an Anvil to rename items with rich colors and styles.
  - **Hex Support**: Directly input hex color codes (e.g., `#FF5555`).
  - **Legacy Codes**: Supports classic `&` color codes (e.g., `&c`).
  - **Gradient System**: Create stunning gradients (e.g., `&q3#FF0000-#0000FF`).
  - **Split Dyeing**: Support for multi-segment color partitioning.
- **Soulbound Enchantment**: Protect your precious gear!
  - Items with this enchantment will remain in your inventory upon death instead of dropping.

## System Adjustments
- **Anvil Enhancement**: Deeply refactored the anvil naming logic to perfectly support Color Sponge formatting.
- **Custom Creative Tab**: Added "Tuanzi's Mod" tab, containing all mod items and pre-enchanted books for easy access.

## Integrations
- **JEI (Just Enough Items)**: Full support!
  - View detailed usage instructions and various dyeing examples for the Color Sponge within JEI.
- **ModMenu**: Basic integration for the Fabric mod list.
